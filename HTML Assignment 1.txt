					Getting Started With HTML Assignment



Ques:1 Write a simple program in HTML that displays the heading “HTML defines the content and structure of your website” on the web browser?

Ans:	<html>
	<head>
	<title>html assignment</title>
	</head>
	<body>
	<h1>HTML defines the content and structure of your website</h1>
	</body>
	</html>

Ques:2 Explain the purpose of comments in HTML and provide an example of how to use comments in an HTML document?
Ans: We use comment tags in HTML so that we could  know and understand about HTML Code. These comments are not visible in the browser.
     Example:-
        <html>
	<head>
	<title>html assignment</title>
	</head>
	<body>
	<!-- <p>HTML Assignment Test</p>-->
	<p>HTML defines the content and structure of your website</p>
	</body>
	</html>

Ques:3 Write an HTML program that includes a heading, a paragraph of text, a horizontal line, and a line break. Arrange these elements to create a simple web page layout?

	</head>

Ques:4  Write a short note on Tag and element with an example?
 Ans:   Tag:- This is known as reserved word and Keyword. Which is made to do special work.It is displayed by angular brackets(<>).
	        <>  - Open Tag
	        </> - Close Tag
        	Example: <p>,<br>,<h1>

	Element:- Tag whatever content is between the start tag and ending tag.That statements are called element.
                  Example:- <i>This is Italic tag</i>

Ques:5  what is the DOCTYPE declaration in HTML?
Ans:  DOCTYPE HTML is a declaration that tells the browser in which version of html the document is written.
